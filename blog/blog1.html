Atribut HTML

  1. Atribut a href (Link)
    Contoh penulisan code :
    <a href="https://github.com/dndyprd" > Google </a>
    Hasilnya : Click Here
  2. Atribut img src (Menampilkan gambar)
    Tambahkan gambar terlebih dahulu pada folder. Contoh penulisan code :
    <img src="path/nama_file.png" alt="Keterangan" ></img>
    Hasilnya :
  3. Style Attribute
    Contoh penulisan code : <p style="color: red">Paragraf Merah</p>
    Hasilnya :

    Paragraf Merah

  4. Title Attribute
    Memberikan keterangan pada elemen HTML saat diarahkan cursor. Contoh penulisan code :
    <p title="Aku judul">Ini adalah paragraf</p>
    Hasilnya :

    Ini adalah paragraf

    <b>Kecamatan Abiansemal</b>

    Contoh penulisan code :

    Contoh penulisan code :

HTML Quotation

  1. Bloqute
    Contoh penulisan code :
    <bloqute>Kabupaten Badung</sup>
    Hasilnya :

    Kata-kata hari ini:

    Tetaplah semangat menjalani hidup, meskipun hidup banyak tantangan. Jangan menyerahh !!
  2. Work Title
    Contoh penulisan code :
    <p><cite>Tetaplah semangat menjalani hidup,</cite> meskipun hidup banyak tantangan. Jangan menyerahh !!</p>
    Hasilnya :

    Tetaplah semangat menjalani hidup, meskipun hidup banyak tantangan. Jangan menyerahh !!

  3. Bi-Directional Override
    Contoh penulisan code :
    <bdo>Aku suka makan bakso!</bdo>
    Hasilnya :

    Aku suka makan bakso

  4. Contact Information
    Contoh penulisan code :
    <address>Jimbaran, Kec. Kuta Utara, Kabupaten Badung, Bali</address>
    Hasilnya :
    Jimbaran, Kec. Kuta Utara, Kabupaten Badung, Bali

HTML CSS

  1. Inline CSS
    CSS yang langsung diketikan didalam tag HTML. Contoh penulisan code :
    <p style="font-size: 24px">AKU SUKA CODING</p>
    Hasilnya :

    AKU SUKA CODING

  2. Internal CSS
    CSS yang masih didalam HTML tapi dibuat dalam tag <style>. Contoh penulisan code :
    <style>
        .internal-css{
            color: red;
            font-size: 24px;
        }
    </style>
    Hasilnya :

    Internal CSS

  3. External CSS
    CSS yang dibuat dalam file terpisah. Contoh penulisan code :
    <link rel="stylesheet" href="style.css">
    Hasilnya :

    External CSS

HTML Link

  1. Target Blank
    Membuat page baru pada browser. Contoh penulisan code :
    <a href="../index.html" target="_blank">Pindah halaman</a>
    Hasilnya : Pindah halaman
  2. Email address
    Contoh penulisan code :
    <a href="mailto:dandypradnyana@gmail.com">Kirim Email</a>
    Hasilnya : Kirim Email

  1. Favicon Contoh penulisan code :
    <link rel="icon" type="image/x-icon" href="/images/favicon.ico">
    Hasilnya : Lihat di pojok kiri atas browser
  2. Page TItle Contoh penulisan code :
    <title>Dandy Blog</title>
    Hasilnya : Lihat di pojok kiri atas browser

HTML Table

  1. Table Header
    Contoh penulisan code :
    <table>
        <tr>
            <th>Nama</th>
            <th>NIM</th>
        </tr>
    </table>
    Hasilnya :
    Nama NIM
  2. Table Row
    Contoh penulisan code :
    <table>
        <tr>
            <td>Dandy</td>
        <tr>
        </tr>
            <td>2415354064</td>
        </tr>
    </table>
    Hasilnya :
    Nama NIM
    Dandy 2415354064
  3. Table Colspan
    Contoh penulisan code :
    <table>
        <tr>
            <th colspan="2">Nama</th>
            <th>Umur</th>
        </tr>
        <tr>
            <td>Dandy</th>
            <td>Pradnyana</th>
            <td>20</th>
        </tr>
    </table>
    Hasilnya :
    Nama Umur
    Dandy Pradnyana 20

HTML List

  1. Unordered List
    Contoh penulisan code :
    <ul>
        <li>Bakso Sapi</li>
        <li>Ayam Goreng</li>
    </ul>
    Hasilnya :
    • Bakso Sapi
    • Ayam Goreng
  2. Ordered List
    Contoh penulisan code :
    <ol>
        <li>React Js</li>
        <li>Laravel</li>
    </ol>
    Hasilnya :
    1. React Js
    2. Laravel
  3. Definition List
    Contoh penulisan code :
    <dl>
        <dt>Pemrograman Website</dt>
        <dd>HTML, CSS, JavaScript</dd>
        <dd>PHP, MySQL</dd>
        <dt>Pemrograman Mobile</dt>
        <dd>Android, Flutter</dd>
        <dd>Apple</dd>
    </dl>
    Hasilnya :
    Pemrograman Website
    HTML, CSS, Javascript
    PHP, MySQL
    Pemrograman Mobile
    Android, Flutter
    Apple
