Tentang Saya

Halo saya Dandy Pradnyana, seorang mahasiswa Teknologi Informasi di Politeknik Negeri Bali. Saya memiliki minat yang besar dalam pengembangan perangkat lunak khususnya pada pembuatan website dan juga game development. Saat ini saya sedang bergabung sebagai tim publikasi di jurusan Teknologi Informasi dan juga menjadi bagian dari Himpunan Mahasiswa Jurusan Teknologi Informasi Politeknik Negeri Bali.

Saya memiliki pengalaman dalam menggunakan berbagai bahasa pemrograman seperti HTML, CSS, JavaScript, PHP, Python, dan juga beberapa framework seperti Tailwind, Laravel dan React. Selain itu, saya juga memiliki pengalaman dalam pengembangan game menggunakan Construct 3 dan Unity. Saya juga mendapatkan sertifikasi "Junior Programmer" dari hasil Ujian Kompetensi Nasional (UKK) yang diadakan oleh LSP SMK Negeri 1 Denpasar dengan status "Sangat Kompeten".
